# foodbot - cкрипт уведомлений о школьном меню в Telegram

Подходит для российских школ, публикующих меню в типовом формате в виде xlsx-файлов по адресу https://school-site/food/YYYY-MM-DD-sm.xlsx, где YYYY-MM-DD - текущая дата.

Скрипт скачивает меню в xlsx-формате с сайта школы, разбирает его согласно ожидаемому формату и отправляет текстовое сообщение в указанный чат Telegram.
Имеется защита от повтороного запуска в тот же день.

## Установка

* Скачать рабочую копию скрипта и перейти в неё

```
git clone https://github.com/ev-smolin/foodbot.git
cd foodbot
```

* Создать virtualenv, например в каталоге env

```
virtualenv env
```
* Установить зависимости командой

```
env/bin/pip install -r requirements.txt
```
* создать в каталоге `project/foodbot` конфигурационный файл `foodbot-config.json` согласно примеру ниже, указав свои значения для url, bot_url, chat_id и message_thread_id
* Добавить команду запуска в `/etc/crontab` в нужное время, например

```
02 7    * * *   user     cd /path/to/foodbot/project/foodbot && ../../env/bin/python __init__.py
```

## Конфигурация:

Скрипт читает конфигурационный файл `project/foodbot/foodbot-config.json`

Пример файла конфигурации:

```
{
    "source": {
        "url": "https://school-site/food/{0}-sm.xlsx",
        "user_agent": "Mozilla/5.0 (X11; Linux x86_64; rv:142.0) Gecko/20100101 Firefox/142.0",
        "ignore_lines": ["итого", "всего"]
    },
    "telegram": {
        "bot_url": "https://api.telegram.org/botNNN:TOKEN",
        "chat_id": "-NNNNNN",
        "message_thread_id": null,
        "message_template": "*Доброе утро!* Сегодня, {0}, в школе приготовят:\n{1}"
    }
}
```

## Запуск

```
cd project/foodbot
../../env/bin/python \_\_init\_\_.py
```

## Настройка в Telegram

* Запустите бота @BotFather и выберите или создайте нового бота
* Настройте имя пользователя
* Скопируйте API token и подставьте его в конфигурационный файл в директиву telegram -> bot_url вместо NNN:TOKEN после строки https://api.telegram.org/bot
* Начните личный чат с ботом или добавьте его в нужную группу, куда должны приходить уведомления
